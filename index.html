Differences between 'switch' and 'if/else'

First, let's define 'switch' and 'if/else':

if/else: When using 'if/else', you're telling the program that if the statement was true then continue down the program, but if the statement was not true then execute the 'else' statement. Once a statement is true, the program will exit the comparison. if/else will check for values in a range.

switch: A switch statement will run through the list of cases looking for a match, and will compare to every case statement until it hits a break. It will also only check data that is prespecified and equal to.

When you would use one or the other: Using 'if/else' is the better option when checking for values in a range and is ideal when trying to select between two options. Using 'switch' is the better option when there's multiple options and you need the statement to spit out a value that matches.

Using '=' vs '==' vs '.equals'

What does each of these do?

=: '=' is used to assign values to variables.

==: '==' is used to compare variables to a constant to check for equality. Typically used for conditional statements. Can only be used with integers and doubles.

.equals: This will compare a string to another string and will only allow true or false. '.equals' is the best way to compare Strings for equality.

Math.random()

What is Math?

Math allows a program or application programmed in Java to execute various mathematic operations.

What is the return type of the ‘.random()’ method?

The typical return type for the '.random()' method is a double between 0.0 and 1.0.

Why is the 'M' in 'Math' capitalized?

The 'M' is capitalized since 'Math.random()' is a class. It is also proper etiquette for other programmers in order to more easily read your code.
